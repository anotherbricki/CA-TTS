## About

CA-TTS, short for **Confidence-Aware Test-Time Scaling**, is a framework for improving multimodal large language models by linking visual perception, confidence calibration, and answer accuracy.

The official implementation includes the codebase, configuration files, benchmark scripts, evaluation utilities, and usage instructions.
